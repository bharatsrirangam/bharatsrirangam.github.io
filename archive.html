MediCheck

MediCheck is an app created for the Amazon Echo/Dot. Its purpose is to help patients with Alzheimer’s Disease and elderly patients with taking their medications by using Amazon’s Alexa to set checkpoints/reminders for these patients. I worked on this project with three friends at the ArchHacks Hackathon in 2016.

CourseTree

CourseTree is a web application that is meant to complement applications such as courseoff.com and Coursicle. It is often difficult for students, especially freshmen, to choose the appropriate courses and which sequential courses to take. CourseTree provides a way for students to conveniently choose and plan for future schedules. I am currently working on this website with a friend, Aiyappa Kodendera. Our contact information and progress can be seen at this link . We update the "dev" branch with our code as we go along and push very final versions of code to the master branch. Prototypes stay in the dev branch.

boRats.Inc

BoRats.Inc is a mobile android application that keeps track of rat sightings throughout New York City. This java app was made in our CS2340 class which aims at giving us experience with approaches to software development in the industry. I was elected Team Leader and assigned work appropriately. I held us accountable to achieve every milestone in time. We developed the application in Java using Android Studio. I implemented Google’s Firebase for multiuser backend capability and tested many of the methods with JUnit classes for quality and functionality. This project was also developed using Agile Principles and Values/Scrum. You can check out the code on our repository. Feel free to follow the README.md to see the application for yourself and to reference all the group members.

If you have any questions or concerns, please contact me at bharatsrirangam@gatech.edu

Neural Network HyperPlanes

I took an Introduction to Artificial Intelligence course the first semester of my sophomore year and in order to get some practice with Neural Networks, I wrote my own hyperplane generation neural net. My program runs a series of randomly generated points to create two sets of points that are linearly separable and then returns the equation to a plane that separates the two sets. This is done with the single perceptron algorithm. You can see the code and run the code from this repository. Feel free to use the code as you see fit.
