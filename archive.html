boRats.Inc

BoRats.Inc is a mobile android application that keeps track of rat sightings throughout New York City. This java app was made in our CS2340 class which aims at giving us experience with approaches to software development in the industry. I was elected Team Leader and assigned work appropriately. I held us accountable to achieve every milestone in time. We developed the application in Java using Android Studio. I implemented Google’s Firebase for multiuser backend capability and tested many of the methods with JUnit classes for quality and functionality. This project was also developed using Agile Principles and Values/Scrum. You can check out the code on our repository. Feel free to follow the README.md to see the application for yourself and to reference all the group members.

Neural Network HyperPlanes

I took an Introduction to Artificial Intelligence course the first semester of my sophomore year and in order to get some practice with Neural Networks, I wrote my own hyperplane generation neural net. My program runs a series of randomly generated points to create two sets of points that are linearly separable and then returns the equation to a plane that separates the two sets. This is done with the single perceptron algorithm. You can see the code and run the code from this repository. Feel free to use the code as you see fit.

HackGT5 Project: Location Scores

LocationScores is a consulting application that won 1st place for the “Best Use of ESRI’s API” at GT’s 5th Hackathon. The application used correlated wealth distribution, employment rate, etc. with existing store performances to find new optimal store locations across the United States. Application was developed using Python, Tensorflow, Keras and Pythons Requests library for efficient data collection and analysis. The actual algorithm involved was a original neural network model that took in demographic data to predict the performance of a store that would be in that location.

Facebook Messenger Analysis Engine

Have you ever wondered how many messages you've sent to a friend on Facebook? Maybe how many messages have been sent in a group chat? Perhaps it was more specific, such as the number of conversations or what types of conversations you've had over the last X number of months. This project, dubbed Insight, was focused on trying to provide this information to me based on my own personal messages. It is implemented in a way that anyone can supply the information of their personal messages and receive the same statistics and metric information. This project was built using Python and attemps to be as scalable as possible as I think of more and more pieces of information that would be useful to keep track of. I am also in the process of using NLP packages such as gensim and nltk to try and determine the topics of specific conversations.
