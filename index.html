Experience

Amazon.com Inc. Internship (Big Data Technologies): This past summer I interned at Amazon's headquarters in Seattle, Washington. I was a Software Development Engineering Intern on the Big Data Technologies. During my internship, I got to build an application for an internal service that essentially allowed users to set preferences on website searches and actions. This feature had certain design constraints and goals such as: having a flexible architecture/implementation to allow for future feature development in the next 5 years, reducing search time latency, and augmenting/smoothening the user experience on the website. I implemented an end to end server-less solution with native AWS to implement my project. In order to create a service for preferences, I created a Restful API service with data analysis features and complete with JUnit tests and full documentation to allow for future development on my project. I also analyzed API request logs with the Hierarchal Clustering and OPTICS clustering algorithms to predict user behavior so that preferences could be predetermined. I demoed my deliverables at weekly meetings with my team and had daily SCRUM meetings with my mentor/manager which helped me develop strong communication skills.

Machine Learning Assistant @ Healthcare Robotics Lab: Currently, I work at the Healthcare Robotics Lab on campus where I am researching the potential benefits of Generative Adversarial Neural Networks(GANs). These potential benefits include being able to traing a neural network to develop complete training data sets for various supervised deep learning algorithms. Some of the technologies I use in my research include Python, NumPy, Keras, and Tensorflow.

CS-2110 Computer Organization and Programming TA: I am currently a teaching assistant for the College of Computing in CS-2110. I get to lead/prepare recitations where we review material from lecture with students and then provide practice assignments to reinforce the ideas. I also create, review, and grade homework assignments and quizzes for the class. Finally, I also organize office hours for students who have unanswered questions.

Georgia Tech College of Computing Tutor: I tutored CS students in some of the fundamental Computer Science courses offered at Georgia Tech such as Object Oriented Programming, Data Structures and Algorithms and Computer Organization and Programming.

Actifio.Inc Internship: I interned at Actifio Inc. where I gained a experience with industry level code, basic software development experience in Java. I was on the UI-platform team which was responsible for creating the REST API for our UI team. I got to learn about basic RestApi structure, SQL databases and queries. I also created test suites with the TestNg Framework. Other developer tools I used include Apache ANT, Hibernate, and PostgreSQL.
